Rustalink features a high-fidelity DSP engine with `f32` precision, ensuring crystal-clear audio manipulation.

---

## DSP Engine Features

| Feature | Impact |
| :--- | :--- |
| **32-Bit Floating Point** | Eliminates integer quantization noise |
| **Low Latency** | Near-real-time filter application |
| **Extensible** | Support for custom plugin filters |
